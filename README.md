GHOSTRACE offers several easy tools on its main screen. Here’s the common way to use it:

### Username Discovery

1. Enter the username you want to search for.  
2. Click the **Search** button.  
3. Wait. The program will look through dozens of platforms.  
4. See results where the username appears.

### Email Breach Check

1. Enter your email address in the Email field.  
2. Click **Check Breach**.  
3. The program checks known data breaches to see if your email appears.  
4. View results on screen.

### Subdomain Enumeration

1. Enter a website address (e.g., example.com).  
2. Click **Find Subdomains**.  
3. GHOSTRACE lists subdomains found for that website.
