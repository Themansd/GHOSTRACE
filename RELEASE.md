## Features

### Core Features
- ✅ Username scanning across 180+ platforms
- ✅ Email breach checking (HaveIBeenPwned integration)
- ✅ WHOIS domain lookup
- ✅ Subdomain enumeration
- ✅ SSL certificate analysis
- ✅ Scan history tracking

### Interface Options
- ✅ Terminal Interface (CLI) with ASCII banner using pyfiglet
- ✅ Modern Web Interface (Flask)
- ✅ Launcher to choose between interfaces

### Technical Features
- ✅ Async scanning for speed
- ✅ User-Agent rotation
- ✅ Rate limiting
- ✅ Concurrent request handling
- ✅ Results saved to JSON files
- ✅ Dark theme UI

## Installation

```bash
pip install -r requirements.txt
```

### Requirements
- Python 3.12+
- aiohttp
- aiodns
- colorama
- python-whois
- flask
- pyfiglet

### Terminal

